# Billing

> Manage credits, top up your balance, and track API spending.

The API uses a prepaid credit balance. Each generation draws from that balance, so you can track cost against output as you go.

## Top up credits

Add credits to your account on the [billing page](https://getimg.ai/app/developer/billing). Click **Top up balance** to make a one-time purchase.

## Auto top-up

On the same [billing page](https://getimg.ai/app/developer/billing), you can enable auto top-up. When your balance drops below a threshold, credits are added automatically.

## Track spending

Monitor your balance and generation usage in the [developer dashboard](https://getimg.ai/app/developer) so you can stay ahead of production spend.

## API billing vs consumer plans

API credits are separate from the consumer plans listed at [getimg.ai/pricing](https://getimg.ai/pricing).

* Credits purchased for the API do not transfer to consumer plans, and vice versa.
* A consumer subscription is not required to use the API.
* The two systems use independent balances.
