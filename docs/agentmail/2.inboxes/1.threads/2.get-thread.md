# Get Thread

GET https://api.agentmail.to/v0/inboxes/{inbox_id}/threads/{thread_id}

**CLI:**
```bash
agentmail inboxes:threads get --inbox-id <inbox_id> --thread-id <thread_id>
```

Reference: https://docs.agentmail.to/api-reference/inboxes/threads/get
