# List Messages

GET https://api.agentmail.to/v0/inboxes/{inbox_id}/messages

**CLI:**
```bash
agentmail inboxes:messages list --inbox-id <inbox_id>
```

Reference: https://docs.agentmail.to/api-reference/inboxes/messages/list
