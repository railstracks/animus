# List API Keys

GET https://api.agentmail.to/v0/inboxes/{inbox_id}/api-keys

**CLI:**
```bash
agentmail inboxes:api-keys list --inbox-id <inbox_id>
```

Reference: https://docs.agentmail.to/api-reference/inboxes/api-keys/list
