# Delete List Entry

DELETE https://api.agentmail.to/v0/lists/{direction}/{type}/{entry}

**CLI:**
```bash
agentmail lists delete --direction <direction> --type <type> --entry <entry>
```

Reference: https://docs.agentmail.to/api-reference/lists/delete
