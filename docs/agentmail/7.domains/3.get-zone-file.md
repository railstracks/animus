# Get Zone File

GET https://api.agentmail.to/v0/domains/{domain_id}/zone-file

**CLI:**
```bash
agentmail domains get-zone-file --domain-id <domain_id>
```

Reference: https://docs.agentmail.to/api-reference/domains/get-zone-file

import Foundation
