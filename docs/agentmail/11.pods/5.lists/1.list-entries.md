# List Entries

GET https://api.agentmail.to/v0/pods/{pod_id}/lists/{direction}/{type}

**CLI:**
```bash
agentmail pods:lists list --pod-id <pod_id> --direction <direction> --type <type>
```

Reference: https://docs.agentmail.to/api-reference/pods/lists/list
