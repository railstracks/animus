# Create List Entry

POST https://api.agentmail.to/v0/pods/{pod_id}/lists/{direction}/{type}
Content-Type: application/json

**CLI:**
```bash
agentmail pods:lists create --pod-id <pod_id> --direction <direction> --type <type> --entry user@example.com
```

Reference: https://docs.agentmail.to/api-reference/pods/lists/create

  }
})
