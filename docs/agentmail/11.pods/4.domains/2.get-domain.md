# Get Domain

GET https://api.agentmail.to/v0/pods/{pod_id}/domains/{domain_id}

**CLI:**
```bash
agentmail pods:domains get --pod-id <pod_id> --domain-id <domain_id>
```

Reference: https://docs.agentmail.to/api-reference/pods/domains/get
