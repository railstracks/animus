# Get Draft

GET https://api.agentmail.to/v0/pods/{pod_id}/drafts/{draft_id}

**CLI:**
```bash
agentmail pods:drafts get --pod-id <pod_id> --draft-id <draft_id>
```

Reference: https://docs.agentmail.to/api-reference/pods/drafts/get
